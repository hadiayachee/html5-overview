HTML5 Overview



HTML5 is an advanced version of HTML. It contains all the features of HTML along with new tags like <video>, <audio> and <canvas>.
There are also tags for including semantics, and these tags are <article>,<header>,<footer> etc.
HTML5 improves the HTML to support the latest multimedia and new features, including graphics, music, etc.
This page will introduce you to :
  • basic structure of HTML5
  • some basic HTML5 tags
Note: if the web browser does not recognize any specific tags, formats, or settings, the browser will render the content based on its default styles.
